border 设置对象边框的特性

  • 语法：border：length style color
  • style：
    • none
    • hidden
    • dotted
    • dashed
    • solid
    • double
    • groove
    • ridge
    • inset
    • outset

border-radius 让这个外边框变成了一个“圆角”的矩形

  • 取值为数字或者百分比

align-items 定义项目在交叉轴上如何对齐

  • 在flex布局的基础上
  • 属性取值：
    • flex-start：交叉轴的起点对齐。
    • flex-end：交叉轴的终点对齐。
    • center：交叉轴的中点对齐。
    • baseline: 项目的第一行文字的基线对齐。
    • stretch（默认值）：如果项目未设置高度或设为auto，将占满整个容器的高度

text-align 元素中的文本行互相之间的对齐方式

  • 默认为取值left
